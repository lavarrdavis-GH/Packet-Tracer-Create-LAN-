# Packet Tracer: Create a LAN

## Project Overview
This project demonstrates the setup of a branch office LAN using Cisco Packet Tracer. The steps include connecting devices, configuring IPv4 addresses, verifying connectivity, and using networking commands.

## Objectives
- Connect network devices and hosts
- Configure devices with IPv4 addressing
- Verify end device configuration and connectivity
- Use networking commands to view host information

## File Structure
- `PacketTracerFiles/`: Contains the Packet Tracer `.pkt` file
- `Instructions/`: Step-by-step instructions for each part of the activity
- `Images/`: Screenshots of device configurations and network topology
- `Notes/`: Addressing tables and supporting documentation

## Usage
1. Open `CreateLAN.pkt` in Cisco Packet Tracer.
2. Follow the instructions in the `Instructions/` folder.
3. Test connectivity between devices and to the internet.
